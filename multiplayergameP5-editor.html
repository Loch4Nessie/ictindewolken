Project 1 - Multiplayer game P5

Wat we gaan doen

Voor het eerste blok willen Dries, Teun, Mees en ik een eigen project doen. We willen een
spel maken met javascript en er voor zorgen dat je dat spel op twee verschillende computers
kan spelen. Ons idee is om de game Pong te maken. We weten niet precies hoelang het duurt om
pong te maken en een server op te zetten, maar als we tijd over hebben willen we het spel verbeteren
door iets van power-ups of een menu toe te voegen aan het spel. We hebben allemaal al kennis over javascript
en P5.js dus we zullen niet heel veel hulp nodig hebben. We zullen ook gebruik maken van tutorials
op Youtube. De rolverdeling voor het maken van Pong online gaat zo: 2 van ons gaan bezig met het
maken van pong zelf in p5.js. De andere twee gaan dan de server opzetten. waarschijnlijk gaat het
opzetten van de server sneller dan het maken van Pong. Als de server is opgezet kan dat team
de code die het andere team heeft gemaakt al in de server te gooien.
